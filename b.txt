Щоб зв'язати функціонал чекбоксів з вашою таблицею даних,
 вам потрібно спочатку надати кожному рядку таблиці чекбокс,
  щоб користувач міг обрати, який рядок він хоче відобразити.
   Для цього ви можете вставити такий код в цикл, який відображає дані:


<td><input type="checkbox" class="form-check-input select-checkbox" value="<?php echo $row['id']; ?>"></td>
Я використовую значення ID як значення чекбоксу.
Вам потрібно також надати вашому кнопці "OK" ID,
щоб ви могли обробити дані, які вибрані користувачем.
Ваш код може виглядати наступним чином:


<button class="btn btn-sm btn-primary badge" id="ok-btn">OK</button>
Після цього ви можете створити функцію,
яка буде виконуватися при натисканні кнопки "OK".
Ця функція отримує всі вибрані чекбокси, і ви можете виконати дії з кожним вибраним рядком таблиці.
Наприклад, ви можете видалити вибрані рядки або оновити їхній стан у базі даних.
Ось приблизний код, який можна використовувати для вибору всіх вибраних рядків:

    javascript
Copy code
$(document).on('click', '#ok-btn', function() {
  var selectedRows = [];
  $('.select-checkbox:checked').each(function() {
    selectedRows.push($(this).val());
  });
  // Do something with the selected rows
});
Зверніть увагу, що ми використовуємо клас "select-checkbox", щоб визначити всі наші чекбокси. Ви можете використовувати будь-який інший клас або ідентифікатор, який вам подобається. Крім того, ми використовуємо метод "each", щоб перебрати всі вибрані елементи та додати їх значення до масиву. Ви можете використовувати будь-які дії з масивом, які вам потрібні, для обробки даних.
